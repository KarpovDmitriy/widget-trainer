# Widget Trainer

## Description

Widget Trainer is an interactive training tool with visual tasks. The project's goal is to create a convenient tool for interactive learning through problem solving. Users will select a topic to test themselves in and answer questions to complete specific tests. There will be different types of widgets:

1. Multiple-choice questions.
2. Single-choice questions.
3. Questions where students must select whether a statement is true or false.
4. Questions where students must complete a section of code.
5. Questions where students must arrange a sequence of blocks (elements) in the correct order.

## Application Features

The application will have login and registration pages, as well as a test history showing the number of correct and incorrect answers for each test.
